🚀 ШВИДКИЙ СТАРТ (3 КРОКИ):

1️⃣  НАЛАШТУЙТЕ .env

   Відкрийте файл: .env
   
   Заповніть:
   - TELEGRAM_TOKEN = токен від @BotFather
   - ADMIN_ID = ваш ID від @userinfobot
   - OPENAI_API_KEY = (опціонально)

2️⃣  ЗАПУСТІТЬ БОТА

   Варіант A (простий):
   - Подвійний клік на: run.bat
   
   Варіант B (PowerShell):
   - Запустіть: run.ps1
   
   Варіант C (командний рядок):
   - py main.py

3️⃣  ТЕСТУЙТЕ

   Напишіть боту в Telegram:
   /start

Помилка: "ModuleNotFoundError"
→ py -m pip install -r requirements.txt

→ Перевірте .env файл

Скріншоти не беруться
→ Встановіть Tesseract OCR
